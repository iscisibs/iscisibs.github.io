      About

      This is a searchable directory of the Big Sib program of the Integrated Science (iSci) undergraduate program at McMaster University. You can learn more about iSci and the Big Sib program here.

      The project is based on the Mathematics Genealogy Project, and data made available by the iSci Society.

      To submit data corrections or request changes, please open an issue on GitHub or get in touch with one of the people below.

      This site is made possible by contributions from Kamea Bellows, Vikram Arora, and Ben Mazin.
